# 1yearBackupGEArchives
Программа для периодического бэкапа архивов в GE Proficy Historian.

Настройка программы происходит в App.config (в папке с бинарным файлом ConsoleHistorian.exe.config).

У нас имеются три параметра, которые отвечают за следующее:
- "DataStoreName". В value указывается хранилище, на которое нужно смотреть.
- "NeedBackup". Новый параметр, если указано "true", значит принудительно вызываем функцию IArchives.Backup. Она берет имя из название файла, заменяет .iha на _backup.zip и готово.
- "BackupInRemove". Если указано "true", то при вызове IArchives.Remove флаг показывает, что нужно сделать бэкап файла. Иногда сам сервер даже с этим флагом этого не делает, поэтому введен флаг выше.
Если нужно отключить что-то конкретное, то выставляем нужные флаги в false.
Лог файл пишется, там можно посмотреть, что программа делает и по какой причине не удалось выполнить то или иное действие.
